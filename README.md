<h1> CI&T - Backend com Java & AWS - Abstraindo um Bootcamp Usando Orientação a Objetos em Java </h1>

**Pré-Requisitos**

- Conhecer a sintaxe da Java
- Java JDK 11
- IDE para desenvolvimento Java
- Git

## Passo a passo


<ul>
  <li><strong>Vamos ABSTRAIR o DOMÍNIO Bootcamp e MODELAR seus ATRIBUTOS E MÉTODOS</strong></li>
  <li><strong>Criaremos as CLASSES: Bootcamp, Cursos, Mentorias e Devs e vamos relaciona-las</strong></li>
  <li><strong>As CLASSES Curso, Mentoria e Devs também serão MODELADOS, ou seja, criaremos seus ATRIBUTOS E MÉTODOS</strong></li>
  <li><strong>Para que o código fique mais legível e de fácil manutenção, iremos utilizar de algumas das ferramentas que o PARADIGMA DE ORIENTAÇÃO A OBJETOS (POO) nos oferece: ABSTRAÇÃO, ENCAPSULAMENTO, HERANÇA E POLIMORFISMO</strong></li>
  <li><strong>E para representar CLASSES que foram criadas e relacionadas, iremos transforma-las em OBJETOS</strong></li>
</ul>
